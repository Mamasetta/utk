Заполнение текстовой страницы сайта

Обратите внимание на оформление данного пункта в левом меню: его название разбито на две строки. Это корректное отображение. Оформление в одну строку выглядело бы неэстетично.

Приступим к изучению стандартных возможностей системы управления сайта (CMS). Этот пример послужит ориентиром для Вас при заполнении страницы текстовым контентом.

Данная страница создана на основе шаблона «Внутренние», без использования каких-либо дополнительных параметров.

Контент добавлен через текстовый редактор системы управления, который своим функционалом похож на программу Word. В нем так же можно изменять внешний вид статьи: например, выделять жирным важные моменты, подчеркивать предложения, использовать курсор, различные шрифты, цветовое выделение шрифтов и пр.

Таблицы

Текстовый редактор системы управления сайтом позволяет создавать таблицы.

Данная таблица может не иметь видимых границ.

В Параметрах таблицы можно корректировать ширину разделительных полос, количество столбцов и строк,расположение на странице и т.д.
